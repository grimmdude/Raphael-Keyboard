Raphael Keyboard

This is a fully scalable keyboard drawn with the Raphael library. Keys on the keyboard are stored in an array called keyboard_keys. The keys of this array correspond to the appropriate key number; ie keyboard_keys['C0', 'A5', 'Dsharp2'] etc. To highlight a certain key just call the desired key object and add the Raphael fill attribute. Here's the highlighting from above: 

	keyboard_keys.C1.attr("fill", "yellow");
	keyboard_keys.G1.attr("fill", "yellow");
	keyboard_keys.C2.attr("fill", "yellow");
	keyboard_keys.E2.attr("fill", "yellow");
